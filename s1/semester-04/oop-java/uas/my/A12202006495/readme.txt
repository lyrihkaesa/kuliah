
# Lihat dokumentasi di "folder doc".

Catatan:
- Saya menggunakan driver mysql connector terbaru.
- Saya menggunakan konsep MVC (Model, View, Controller)
    - Model = Entity + Repository
    - View = View
    - Controller = Service + Util
- Jika ingin menjalankan aplikasi anda harus membuat database terlebih dahulu.
- Import "backup.sql" pada PHPMyAdmin atau Database klien lainnya.
- Atur Classpath jika menggunakan Visual Studio Code ke folder "project".
- Jalankan program App.java atau FrameLogin.java.
- Selesai

Catatan Java Arcive:
- Buka terminal pada folder yang terdapat ".jar" dan "application.properties"
- Balankan perintah "java -jar application.jar"

Nama File
- Saya menggunakan penamaan yang mudah dimengerti, contoh:
    - Login = FrameLogin
    - FMenu = FrameMenu
    - FPegawai = FramePegawai
    - FBarang = FrameBarang
    - FJual = FrameJual
